# Progetto Ingegneria del Software - Sistema di Carpooling 🚗

### Descrizione del Progetto
Sviluppo di un sistema software completo per la gestione di un servizio di **Carpooling**, volto a facilitare la condivisione di viaggi tra pendolari. Il progetto è stato realizzato seguendo le specifiche d'esame per il corso di Ingegneria Informatica dell'Università degli Studi di Napoli Federico II.

### Funzionalità Implementate (Specifiche d'Esame)
Il sistema soddisfa i seguenti requisiti tecnici descritti nella traccia ministeriale:
* **Registrazione Utenti:** Gestione profili con dati personali, contatti e dettagli veicolo.
* **Pubblicazione e Ricerca:** Inserimento di itinerari, orari e contributi spese; ricerca basata su criteri geografici.
* **Sistema di Prenotazione:** Generazione di biglietti identificativi con ID univoco e dettagli costi.
* **Gestione Autista:** Interfaccia grafica per l'accettazione o la cancellazione delle prenotazioni.
* **Feedback & Reportistica:** Sistema di valutazione reciproca e modulo amministratore per il monitoraggio degli incassi e degli utenti.

### Organizzazione dei File
Seguendo le istruzioni di consegna del corso:
- `/Documentazione`: Relazione tecnica in formato `.pdf` e `.doc`.
- `/UML`: Progetto completo realizzato con **Visual Paradigm** (`.vpp`).
- `/Codice`: Sorgenti Java sviluppati in ambiente **Eclipse**.
